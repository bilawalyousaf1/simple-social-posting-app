# Simple Social Posting App

A minimal frontend social posting application built with vanilla JavaScript, HTML, and CSS. It simulates user authentication and post interactions entirely in the browser using `localStorage`.

## Features

- User signup with email validation and password confirmation
- Login with credential verification
- Protected post feed (accessible only after login)
- Create, edit, and delete your own posts
- Like / unlike any post (visible to all users)
- Real‑time relative timestamps with Moment.js
- SweetAlert2 popups for user feedback
- Logout clears session and redirects to login

## Technologies Used

- HTML5
- CSS3 (custom styling with video background)
- Vanilla JavaScript (ES6+)
- [SweetAlert2](https://sweetalert2.github.io/) – elegant alerts
- [Moment.js](https://momentjs.com/) – relative time formatting
- [Font Awesome](https://fontawesome.com/) & [Bootstrap Icons](https://icons.getbootstrap.com/) – icon set
